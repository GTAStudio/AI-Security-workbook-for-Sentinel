# AI Security Workbook for Sentinel

> English | 中文

## Overview
This repository provides Microsoft Sentinel workbook JSON files for AI security monitoring. It includes an English and a Chinese version of the AI Security workbook, plus an AVD workbook.

## Files
- `AI security workbook en.json`  English AI Security workbook
- `AI security workbook cn.json`  Chinese AI Security workbook (translated)
- `avd workbook cn.json`  AVD workbook (Chinese)

## Usage
1. In Microsoft Sentinel, go to **Workbooks**.
2. Choose **Add workbook**  **Advanced editor**.
3. Paste the JSON content from the file you want to import.
4. Save and pin to your dashboard as needed.

## Notes
- The Chinese workbook focuses on UI labels and visible text translation.
- KQL logic remains intact to preserve query accuracy.

# AI 安全工作簿（Sentinel）

> English | 中文

## 简介
本仓库提供用于 Microsoft Sentinel 的 AI 安全监控工作簿 JSON 文件，包含英文与中文版本的 AI 安全工作簿，以及一个中文 AVD 工作簿。

## 文件列表
- `AI security workbook en.json`  英文 AI 安全工作簿
- `AI security workbook cn.json`  中文 AI 安全工作簿（翻译版）
- `avd workbook cn.json`  中文 AVD 工作簿

## 使用方式
1. 打开 Microsoft Sentinel  **工作簿**。
2. 点击 **添加工作簿**  **高级编辑器**。
3. 粘贴对应 JSON 文件内容。
4. 保存并按需固定到仪表板。

## 说明
- 中文版主要翻译界面可见文本。
- KQL 查询逻辑保持不变以确保准确性。
